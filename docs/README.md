# Tài liệu (react_template)

Thư mục này **không nằm trong project demo** (`react-demo`). Dùng cho bản thân ôn tập; khi demo hoặc gửi repo cho interviewer, chỉ cần mở/share folder **react-demo**, không cần đưa `docs/` vào.

## Nội dung

- **frontend-interview/** — Đề bài phỏng vấn frontend (React), hướng dẫn triển khai và code mẫu. Làm bài trong project `react-demo`, đọc tài liệu tại đây khi ôn.
